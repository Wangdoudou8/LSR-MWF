Description:
1 First, go to compare-mt-master from the terminal, then input and run "pip install-r requirements.txt", then input and run "python setup.py install" to install the code for this version of the ROUGE evaluation metric.
2 Decompress cnndm.zip in the data file and select Decompress to current folder.
3 Run the 11ama3-70B-sum.py command in llm_code.
4 The generated reasoning process is automatically saved to the llm_code/cnndm_rationale folder, and the remaining output is saved to the llm_code/llama3_output_train folder.
